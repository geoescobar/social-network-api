## Installation

##### This is a Node.Js based application. You can run the application using your integrated terminal.

1. Clone repo from GitHub

```
gh repo clone geoescobar/social-network-api
```

2. Install your dependencies

```
npm i
```

3. Start the server

```
npm start
```
